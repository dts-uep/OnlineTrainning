HOMEPAGE
/

UPLOAD
/upload-main: upload the main script/notebook (.py, .ipynb)
/upload-data/{job_id}: upload the data needed for script/notebook (.csv)
/upload-util/{job_id}: upload the utils needed for script/notebook (.py)

FILES
/files/run-script/{job_id}: start running the main script
/files/download/{job_id}: download results zip file once complete

STATUS
/status/result/{job_id}: get the current status of the script running job
/status/full/{job_id}: query for full information

ERROR /error/error-log/{job_id}: get execution error if any
IN SCRIPT NOTE - There should only be one main script
- If there is external module, upload in /upload-util/{job_id}
- In script should be import utils.module
- To get utils folder path:
utils_dir = os.getenv("UTILS_DIR")
utils_path = os.path.join(utils_dir, module)
- If there is data, upload in /upload-data/{job_id}
- To get data folder path:
data_dir = os.getenv("DATA_DIR")
data_path = os.path.join(data_dir, data)
- For any other saved object:
output_dir = os.getenv("OUTPUT_DIR", "/sandbox/workplace")
save_path = os.path.join(output_dir, object_file)
